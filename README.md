# Simple Calculator in Java

## Description

This is a simple command-line calculator written in Java. It performs basic arithmetic operations:

* Addition
* Subtraction
* Multiplication
* Division

## Features

* User-friendly menu
* Handles division by zero
* Uses Java `switch` statement
* Accepts decimal values

## Technologies Used

* Java
* Scanner class

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/simple-calculator.git
```

2. Compile the program:

```bash
javac SimpleCalculator.java
```

3. Run the program:

```bash
java SimpleCalculator
```

## Example

```
Enter first number: 10
Enter second number: 5

Choose operation:
1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)

Enter choice: 1

Result = 15.0
```
